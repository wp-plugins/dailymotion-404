Quelques vidéos d'erreur 404 de Dailymotion à ajouter à votre page "404 Not Found" !

== Description ==

Quelques vid&eacute;os d'erreur 404 de Dailymotion &agrave; ajouter &agrave; votre page "404 Not Found" !

== Installation ==

Il suffit d'installer le plugin dans le répertoire de plugins WordPres, de l'activer, et puis
de mettre dans votre code :

		<?php dailymotion404(); ?>
		
et vous avez les vidéos !
